# 💰 Smart Expense Tracker (Java)

## 📌 Overview
Smart Expense Tracker is a console-based Java application that helps users record, store, and analyze their expenses. It uses file handling for persistent storage and provides useful insights into spending habits.

---

## 🚀 Features
- Add expenses with category and amount
- Store data in a file (expenses.txt)
- View all recorded expenses
- Calculate total spending
- Category-wise expense analysis
- Identify highest expense
- Display top 3 expenses using sorting

---

## 🛠 Tech Stack
- Java
- File Handling (FileWriter, BufferedReader)
- Data Structures (HashMap, ArrayList)

---

## 📂 Project Structure

expenses.txt

## ▶️ How to Run
1. Compile:
javac ExpenseTracker.java


2. Run:
java ExpenseTracker
